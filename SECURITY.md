# Security Policy

## Supported Versions

Cuecast is currently maintained on the `main` branch.

## Reporting a Vulnerability

Please do not open public GitHub issues for suspected security problems.

Report vulnerabilities privately through GitHub's private vulnerability reporting if it is enabled for the repository, or contact the maintainer directly through a private channel.

When reporting an issue, include:

- a short description of the problem
- affected version or commit
- reproduction steps if available
- impact assessment
- any proposed mitigation

I will review reports privately and coordinate a fix before public disclosure when appropriate.
