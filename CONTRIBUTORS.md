# Contributors

## Maintainer

- Cuecast is currently maintained by the repository owner.

## Contributions

Contributions are welcome through pull requests.

Please keep changes aligned with the project's core rules:

- preserve the original stream bytes without transcoding
- keep segment splitting metadata-driven unless a change is explicitly intentional
- avoid committing private station data, personal paths, or service-specific listening details

## Recognition

Contributors may be listed here over time as the project grows.
